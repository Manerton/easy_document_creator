О Сервисе

Easy Document Creator - сервис генерации документов из подготовленных вами шаблонов и данных.

Для работы сервиса достаточно заранее подготовить шаблон документа.

Шаблон должен быть одного из двух форматов DOCX или XLSX.

После чего создаём процесс указывая в качестве шаблона ранее подготовленный документ

Для получения готового документа достаточно выполнить процесс загрузив соответсвующий набор данных

Подробнее о функционале и возвожностях сервиса читайте в документации
